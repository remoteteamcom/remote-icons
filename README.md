## Install

**Install with npm**

```sh
$ npm install remote-icons
```

**Install with Yarn**

```sh
$ yarn add remote-icons
```

# Usage

```html
<span class="remote-icon remote-icon-arrow-down">
</span>
```

# Contribude

* Install dependencies

```
yarn
```

* Change icons in *svgs* folder

* Run npm script

```
yarn build
```
